# M이론

*다섯 끈이론의 어머니 — 11번째 차원에서 드러나는 통일*

---

## 🎯 핵심 질문

왜 10차원 끈이론이 다섯 가지나 있는가? 이들이 모두 옳다면 서로 어떻게 양립하는가? 그리고 이들을 통합하는 단일 이론이 있다면 그 이론은 무엇인가?

1994-1995년의 "두 번째 끈이론 혁명" 이전까지, 물리학자들은 다섯 가지의 완전히 다른 10차원 끈이론이 존재한다는 사실 앞에서 당혹했다: Type I, Type IIA, Type IIB, $\text{Heterotic-SO}(32)$, $\text{Heterotic-}E_8\times E_8$. 이들은 각각 자체적으로 일관적인 양자 이론이었다. Witten(1995)은 이 다섯 이론 모두가 **11차원에서 정의되는 단일 이론**(M이론)의 서로 다른 결합상수 · 기하학적 극한임을 보였다. 11번째 차원은 Type IIA의 강결합 극한에서 창발적으로 나타난다 (→ emergent-spacetime).

---

## 🌍 어디서 나타나나

**다섯 끈이론의 파라미터 공간.** 각 끈이론은 결합상수 $g_s$와 기하학적 파라미터(콤팩트 차원의 크기·모양)를 가진다. 이 파라미터 공간 전체를 하나의 "이론 공간"으로 보면, 각 끈이론은 이 공간의 특정 구석(극한)에 위치한다.

**11차원 초중력.** 11차원에서 초대칭과 일반공변성(general covariance)을 요구하면 **유일한** 이론이 나온다: 11차원 초중력(Cremmer-Julia-Scherk, 1978). 이것이 M이론의 저에너지 유효 이론이다. 11차원 초중력의 보손 섹터:
$$S_{11} = \frac{1}{2\kappa_{11}^2}\int d^{11}x\sqrt{-g}\left(R - \frac{1}{2}|G_4|^2\right) - \frac{1}{6\kappa_{11}^2}\int C_3 \wedge G_4 \wedge G_4$$
여기서 $G_4 = dC_3$은 4-형식 장(field strength), $C_3$은 3-형식 게이지 퍼텐셜이다.

**M이론의 기본 대상.** 점입자나 1차원 끈이 아닌 막(membrane, brane)이 기본 대상이다:
- **M2-브레인:** 2+1차원 부피를 가지는 막. 전기적으로 $C_3$에 결합한다.
- **M5-브레인:** 5+1차원 부피를 가지는 막. 자기적으로 $C_3$의 쌍대 $C_6$에 결합한다.

---

## 🔍 직관의 함정

**함정 1: "M이론은 11차원 끈이론이다"**

✗ 틀렸다. M이론의 기본 대상은 1차원 끈이 아니라 2차원 막(M2-브레인)이다. "막이론(Membrane theory)"이라 부르지 않는 것은 이 이름이 너무 제한적이기 때문이기도 하지만, 무엇보다 완전한 이론의 기본 자유도가 무엇인지 우리가 모르기 때문이다. "M"의 의미는 의도적으로 모호하게 남겨졌다(Witten: "M은 Magic, Mystery, Membrane 중 원하는 것").

**함정 2: "11번째 차원은 공간에 추가된 여분 차원이다"**

더 정확히 말하면, 11번째 차원은 Type IIA 끈이론의 강결합 극한에서 **창발**한다. Type IIA 이론에서 끈 결합상수 $g_s$를 키우면, 기존 10차원에서는 보이지 않던 새로운 차원이 그 크기 $R_{11} \sim g_s^{2/3} \ell_s$로 나타난다. 즉 공간의 차원 수 자체가 결합상수에 의존하는 창발적 특성이다.

**함정 3: "M이론이 완성된 이론이다"**

✗ M이론은 완성되지 않았다. 우리가 알고 있는 것은 그 저에너지 극한(11차원 초중력)과 다섯 끈이론을 그 다양한 극한으로 포함한다는 구조적 사실뿐이다. 완전한 비섭동 정의는 미완 상태다.

**함정 4: "다섯 이론 + M이론 = 여섯 이론"**

다섯 끈이론과 M이론이 별개의 여섯 이론이 아니다. M이론이 하나의 이론이고, 다섯 끈이론은 그 이론의 서로 다른 파라미터 극한에서의 서술이다. "이론의 개수"가 아니라 하나의 이론의 "풍경(landscape)"이다.

## ⚙️ 제1원리 유도

**Type IIA에서 11번째 차원의 창발.** Type IIA 초중력의 결합상수-크기 관계를 추적하자.

Type IIA 끈이론의 파라미터:
- 끈 길이: $\ell_s = \sqrt{\alpha'}$
- 끈 결합상수: $g_s = e^\phi$ (딜라톤 기댓값)

Type IIA의 D0-브레인 질량:
$$m_{D0} = \frac{1}{g_s \ell_s}$$

$g_s \to \infty$ 극한에서 D0-브레인이 가벼워진다. 이 상태들이 Kaluza-Klein 타워를 형성한다면, 이것은 반지름 $R_{11}$인 콤팩트 차원의 KK 모드:
$$m_{\text{KK}}^{(n)} = \frac{n}{R_{11}}$$

D0-브레인이 KK 모드라면(n=1):
$$R_{11} = g_s \ell_s$$

더 정밀하게, 플랑크 단위를 고려하면:
$$R_{11} = g_s^{2/3} \ell_s \cdot g_s^{1/3} = g_s \ell_s$$

11차원 플랑크 길이와 끈 파라미터의 관계:
$$\ell_{11} = g_s^{1/3} \ell_s$$

**11차원 초중력으로의 환원.** M이론을 $R_{11}$인 원 $S^1$에 콤팩트화하면 Type IIA 끈이론이 나온다:
$$g_s = \left(\frac{R_{11}}{\ell_{11}}\right)^{3/2}, \qquad \ell_s = \frac{\ell_{11}^{3/2}}{R_{11}^{1/2}}$$

$R_{11} \to 0$이면($g_s \to 0$) Type IIA 약결합 끈이론. $R_{11} \to \infty$이면($g_s \to \infty$) 완전히 11차원적인 이론, 즉 M이론 자체.

**$E_8 \times E_8$ Heterotic → Horava-Witten.** M이론을 구간(interval) $[0, \pi\rho]$에 콤팩트화하면, 두 끝 경계(boundary)에 각각 $E_8$ 게이지 이론이 등장한다. 이것이 $\text{Heterotic-}E_8\times E_8$ 이론의 M이론적 기원이다(Horava-Witten, 1995):
$$M\text{이론} / (S^1/\mathbb{Z}_2) = \text{Heterotic-}E_8\times E_8$$

**쌍대성 망의 완성.** M이론의 다양한 콤팩트화와 극한으로부터: $\square$

| 극한 | 결과 |
|------|------|
| $R_{11} \to 0$ | Type IIA |
| $R_{11} \to 0$, T-쌍대성 적용 | Type IIB |
| $S^1/\mathbb{Z}_2$ 콤팩트화 | Heterotic $E_8\times E_8$ |
| S-쌍대성 후 $R_{11} \to 0$ | Type I, Heterotic $SO(32)$ |

## 🧪 검증 현실

✗ **직접 실험 검증 없음.** M이론은 실험적으로 검증된 예측을 아직 내놓지 못했다. 이론이 발전하는 과정에서 실험으로 구별 가능한 예측이 나오지 않았다.

✓ **수학적 구조의 내부 일관성.** "검증"은 다음 수학적 자기 일관성 검사로 이루어진다:

- **저에너지 유효 이론의 일치 (✓):** M이론을 $S^1$에 콤팩트화한 후 저에너지 극한을 취하면 정확히 Type IIA 초중력이 나온다. 스펙트럼, 대칭, 상호작용 모두 일치한다.
- **BPS 스펙트럼 일치 (✓):** M2-브레인이 $S^1$을 감으면 Type IIA의 기본 끈이 된다. M2-브레인이 $S^1$의 점에 위치하면 D2-브레인이 된다. M5-브레인이 $S^1$을 감으면 D4-브레인이 된다. M5-브레인이 점에 위치하면 NS5-브레인이 된다. 이 대응이 BPS 질량 공식 수준에서 일치한다.
- **11차원 아노말리 상쇄 (✓):** Horava-Witten 이론에서 경계의 $E_8$ 게이지 아노말리가 벌크 11차원 초중력의 기여로 정확히 상쇄된다. 이는 이론의 내부 일관성을 확인한다.

✗ **비섭동 M이론의 직접 계산 불가.** M이론의 완전한 비섭동 정의가 없으므로, 강결합 영역에서의 직접 계산은 현재 불가능하다.

---

## 🔬 경계

**이것이 M이론의 주요 미완성 영역이다.**

✗ **완전한 비섭동 정의 없음.** 이것이 M이론의 가장 큰 미해결 문제다. 끈이론은 세계면 CFT라는 섭동적 정의를 가지지만, M이론에는 이에 해당하는 비섭동 기초 공식이 없다. M이론은 그 극한들로 정의될 뿐이다.

✗ **M2/M5-브레인의 동역학 이론 미완.** M5-브레인 위의 저에너지 이론은 6차원 $\mathcal{N}=(2,0)$ 이론인데, 이것은 라그랑지안이 알려지지 않은 이론이다. M2-브레인 위의 이론(ABJM 이론)은 최근 발전이 있었지만 완전하지 않다.

✗ **기본 자유도 미확인.** M2-브레인과 M5-브레인 중 어느 것이 더 "기본"인지, 또는 완전히 다른 자유도 집합이 있는지 알 수 없다. Matrix 이론(Banks-Fischler-Shenker-Susskind, 1996/1997)은 M이론의 D0-브레인 양자역학적 기술을 제안했지만, 11차원 로렌츠 불변성이 명시적이지 않은 등의 한계가 있다.

✓ **저에너지 극한(11차원 초중력)은 완전히 알려짐.** M이론의 저에너지 유효이론과 여러 극한들의 구조는 명확히 이해되어 있다.

---

## 🧬 횡단 원리

**Kaluza-Klein 이론.** M이론의 11번째 차원은 Kaluza-Klein 아이디어의 끈이론적 실현이다. KK 이론은 5차원 중력이 4차원 중력 + 전자기력을 준다는 것이었다. M이론에서는 11차원이 10차원 끈이론을 준다.

**막이론의 역사.** Dirac(1962)은 전자를 확장된 객체(막)로 기술하려 했다. 이 아이디어는 끈이론에 앞선 것이었으나 당시 기술적 어려움으로 포기되었다. M이론에서 막이 다시 기본 대상으로 복귀한다.

**Matrix 이론과 비가환 기하학.** M이론의 행렬 이론적 기술에서 D0-브레인의 위치 좌표가 행렬이 된다:
$$[X^\mu, X^\nu] \neq 0$$
이것은 공간 좌표의 비가환성이며, Connes의 비가환 기하학과 연결된다. 플랑크 스케일에서 공간이 연속적 다양체가 아닐 수 있다는 암시다 (→ quantum-gravity).

**AdS/CFT와 M이론.** M2-브레인과 M5-브레인의 근거리 기하학은 각각 $\text{AdS}_4 \times S^7$과 $\text{AdS}_7 \times S^4$를 준다. 이것은 M이론에서의 AdS/CFT 대응의 기초이며, 이 대응에서 M이론적 효과가 경계 이론의 비섭동 보정으로 나타난다.

## 🪜 창발

M이론이 드러내는 창발의 층위는 여러 개다.

**공간 차원의 창발.** 가장 직접적인 예: 11번째 공간 차원은 Type IIA 이론의 결합상수 $g_s$에 따라 "나타나거나 사라진다". $g_s \to 0$이면 10차원 이론, $g_s \to \infty$이면 11차원 이론. 공간의 차원 수 자체가 근본적이지 않고 파라미터의 함수다 (→ emergent-spacetime, L6).

**끈의 비근본성.** 끈이론은 끈을 기본 대상으로 삼는다. M이론에서 끈(Type IIA)은 M2-브레인이 11번째 차원을 감은 결과로 나타난다. 끈 자체가 창발적 대상이다.

**쌍대성 망의 창발적 의미.** 다섯 끈이론이 하나의 이론의 극한이라는 것은, 어떤 이론도 다른 이론보다 더 "근본"이 아님을 의미한다. M이론은 이 대칭성을 명시한다: 모든 기술은 특정 파라미터 극한에서만 유효한 유효 기술이다.

**그러나 M이론 자신도?** M이론이 더 깊은 이론의 특정 극한은 아닐까? 이 물음은 열려 있다. M이론의 완전한 비섭동 정의를 모르는 한, M이론 자신이 창발적인지 아닌지도 알 수 없다.

## 📐 예측·반증

✓ **BPS 스펙트럼 예측.** M이론은 다양한 BPS 상태들의 질량과 전하 관계를 예측한다. 이 예측들은 서로 다른 끈이론 극한들에서 독립적으로 계산되어 일치해야 한다. 이 일치는 이론 내에서 확인된다.

✓ **초중력 극한 예측.** M이론 $\to$ 11차원 초중력 $\to$ 10차원 초중력의 환원 연쇄가 명시적으로 확인된다. 이것은 반증 가능한 수학적 명제들이며 일관성이 검증된다.

✗ **현상론적 예측의 부재.** M이론이 직접 관측 가능한 현상(새로운 입자, 힘, 우주론적 신호)을 예측하는 단계에 아직 이르지 못했다. 이것은 이론의 미완성 상태를 반영한다.

✓ **Matrix 이론 예측.** BFSS Matrix 이론은 특정 과정(D0-브레인 산란)에서 계산 가능한 예측을 낸다. 이 예측들이 11차원 초중력 계산과 일치함이 확인된다(하지만 전체 M이론에 대한 완전한 정의로는 부족하다).

**우주론적 가능성.** Horava-Witten 기하학(두 $E_8$ 브레인 사이의 11번째 차원)은 "Ekpyrotic 우주론"(충돌하는 브레인이 빅뱅을 일으킨다는 가설)의 기반이 되었다. 이것은 원칙적으로 관측 가능한 예측(CMB 스펙트럼)을 낼 수 있지만, 현재 관측 데이터와의 비교에서 확정적 결과가 없다.

3. M이론의 완전한 비섭동 정의를 위한 후보들(Matrix 이론, AdS/CFT, ...)은 어떤 상태인가? → Ch4 브레인, quantum-gravity

---

🧩 **Principle** — 다섯 가지 10차원 끈이론과 11차원 초중력은 단일한 M이론의 서로 다른 파라미터 극한이다. M이론의 기본 대상은 M2/M5-브레인이며, 11번째 차원은 Type IIA 끈이론의 강결합에서 창발한다.

🔬 **Boundary** — M이론의 완전한 비섭동 정의는 아직 존재하지 않는다. 우리는 M이론의 저에너지 극한(11차원 초중력)과 다양한 섭동 극한(다섯 끈이론)을 알지만, 중간 영역과 비섭동 동역학을 기술하는 기본 공식은 미완이다.

🪜 **Emergence** — 공간 차원의 수 자체가 창발적이다: 10차원과 11차원은 동일한 이론의 다른 파라미터 영역이다. 끈도 M2-브레인의 창발적 대상이다. M이론은 무엇이 근본인지에 대한 질문을 해결하는 것이 아니라, 그 질문을 더 깊은 수준으로 밀어넣는다.
