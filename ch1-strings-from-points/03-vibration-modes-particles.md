# 진동 모드 = 입자

*끈의 정규 진동 모드를 양자화하면 각 모드가 하나의 소립자로 나온다 — 무한히 많은 입자의 탑이 하나의 끈에서 비롯된다.*

---

## 🎯 핵심 질문

"소립자는 끈의 진동 모드다"라는 말은 끈이론의 핵심 슬로건이다. 그런데 이것이 정확히 무슨 뜻인가? 기타 줄을 튕기면 배음(harmonic)이 나오는 것처럼, 끈도 진동하면 특정 주파수의 모드가 생긴다. 이 진동 모드 각각이 입자 하나에 대응한다. 하지만 왜 특정 모드가 특정 입자인가? 그리고 질량은 어떻게 결정되는가?

이 문서에서는 끈의 세계면 운동 방정식의 해를 푸리에 전개하고, 각 모드를 독립적인 조화 진동자로 양자화하여, 질량 스펙트럼 공식

$$M^2 = \frac{N - a}{\alpha'}$$

을 유도한다. (→ L1 oscillations 정규모드)

---

## 🌍 어디서 나타나나

끈의 진동 모드와 입자의 대응은 두 가지 맥락에서 중요하다.

**역사적 맥락:** 1968년 베네치아노(Veneziano)는 강력 산란 진폭의 실험 데이터를 설명하는 공식을 발견했다. 1970년 남부(Nambu), 닐센(Nielsen), 수스킨트(Susskind)는 이 공식이 사실 진동하는 1차원 물체, 즉 끈의 산란 진폭임을 인식했다. 이것이 끈 이론의 탄생이다.

**현대적 맥락:** 끈이론이 강력의 이론으로서 포기된 이후에도, 질량 스펙트럼 구조 — 특히 질량이 없는 스핀-2 입자의 필연적 출현 — 가 중력 이론으로 재해석되면서 현재에 이른다. (→ `04-graviton-necessity.md`)

---

## 🔍 직관의 함정

**함정 1: "진동 모드는 무한히 많으니 무한히 많은 입자가 있다."**

정확하다. 끈이론은 무한히 많은 질량 준위를 예측한다. 대부분은 플랑크 질량 $M_P \sim 10^{19}\ \text{GeV}$ 스케일이므로 현재 기술로는 관측 불가능하다. 저에너지에서는 질량이 없는(massless) 몇 가지 모드만 보인다.

**함정 2: "영점에너지 $a$는 그냥 기술적 상수다."**

영점에너지는 각 모드의 양자 영점 진동에서 비롯된다. 그런데 이 합산이 발산한다 — $\sum_{n=1}^\infty n = ?$. 제타 함수 정규화를 쓰면 $\sum_{n=1}^\infty n = \zeta(-1) = -1/12$. 이 결과는 임의적으로 보이지만, 빛뿔 게이지 양자화와 등각장론 두 독립적인 방법으로 같은 답을 준다. $a = 1$ (보소닉, 열린 끈)은 이론 내부에서 고정되는 수다.

**함정 3: "타키온은 이론의 실패다."**

보소닉 끈의 기저 상태($N = 0$)는 $M^2 = -1/\alpha' < 0$으로 **타키온(tachyon)**이다. 이것은 이론이 불안정한 진공 위에 정의되어 있음을 뜻한다 — 마치 멕시코 모자 퍼텐셜의 꼭대기에 서 있는 것처럼. 그러나 이것은 보소닉 끈 이론의 문제이지 끈 이론 전체의 실패가 아니다. 초끈 이론은 GSO 사영(projection)으로 타키온을 제거한다.

## ⚙️ 제1원리 유도

### 운동 방정식의 해: 모드 전개

이전 문서 (→ `02-string-action.md`)에서 등각 게이지의 운동 방정식이 2차원 파동 방정식임을 보였다:

$$(\partial_\tau^2 - \partial_\sigma^2) X^\mu = 0$$

이 방정식의 일반해를 **좌행파**와 **우행파**의 합으로 쓴다:

$$X^\mu(\tau, \sigma) = X^\mu_L(\tau + \sigma) + X^\mu_R(\tau - \sigma)$$

#### 열린 끈 (노이만 경계 조건)

$\partial_\sigma X^\mu|_{\sigma=0,\pi} = 0$를 사용하면 좌행파와 우행파가 섞이며:

$$
X^\mu(\tau, \sigma) = x^\mu + \frac{\alpha'}{1} p^\mu \tau + i\sqrt{\frac{\alpha'}{2}} \sum_{n \neq 0} \frac{\alpha^\mu_n}{n} e^{-in\tau} \cos(n\sigma)
$$

여기서 $x^\mu$는 질량 중심 위치, $p^\mu$는 전체 운동량, $\alpha^\mu_n$은 진동 모드 계수다.

#### 닫힌 끈 (주기적 경계 조건)

$X^\mu(\tau, \sigma + 2\pi) = X^\mu(\tau, \sigma)$를 사용하면 좌행파와 우행파가 독립적으로:

$$
X^\mu = x^\mu + \alpha' p^\mu \tau + \frac{i}{2}\sqrt{\frac{\alpha'}{2}} \sum_{n \neq 0} \left[\frac{\alpha^\mu_n}{n} e^{-2in(\tau-\sigma)} + \frac{\tilde{\alpha}^\mu_n}{n} e^{-2in(\tau+\sigma)}\right]
$$

$\alpha^\mu_n$은 우행 모드, $\tilde{\alpha}^\mu_n$은 좌행 모드다. 닫힌 끈은 두 세트의 독립적인 진동자를 가진다.

### 양자화: 포아송 괄호 → 교환자

정준 양자화를 수행한다. 고전적 포아송 괄호 $\{X^\mu(\sigma), \Pi_\nu(\sigma')\} = \delta^\mu_\nu \delta(\sigma - \sigma')$를 교환자로 올리면:

$$[\hat{X}^\mu(\sigma), \hat{\Pi}_\nu(\sigma')] = i\delta^\mu_\nu \delta(\sigma - \sigma')$$

모드 계수에 대해서:

$$[\alpha^\mu_m, \alpha^\nu_n] = m\, \delta_{m+n,0}\, \eta^{\mu\nu}$$

이것은 각 모드가 독립적인 **조화 진동자**처럼 행동함을 의미한다. $\alpha^\mu_{-n}$ ($n > 0$)은 생성 연산자, $\alpha^\mu_n$ ($n > 0$)은 소멸 연산자다. (→ L1 oscillations)

### 비라소로 연산자와 질량 공식

비라소로 구속 조건 $T_{\pm\pm} = 0$을 모드로 전개하면 비라소로 연산자 $L_m$이 된다. 특히 $m = 0$:

$$
L_0 = \frac{\alpha'}{4} p^2 + \sum_{n=1}^\infty \alpha_{-n} \cdot \alpha_n
$$

구속 조건 $L_0 = a$ (정규순서화 후 영점에너지 $a$를 보정)를 쓰면:

$$\frac{\alpha'}{4} M^2 = N - a$$

여기서 $N = \sum_{n=1}^\infty \alpha_{-n} \cdot \alpha_n$은 **수(Number) 연산자**이며, $M^2 = -p^2$은 입자의 질량 제곱이다. 열린 끈에 대해:

$$
\boxed{M^2 = \frac{N - a}{\alpha'}}
$$

닫힌 끈에 대해서는 좌행/우행 각각에 $L_0 = \tilde{L}_0$인 수준 정합 조건이 추가된다:

$$M^2 = \frac{4(N - a)}{\alpha'} = \frac{4(\tilde{N} - \tilde{a})}{\alpha'}$$

영점에너지 $a$의 계산: 각 보손 자유도는 영점에너지 $+1/2$를 기여하고, 정규화하면 다음이 나온다. $D - 2$개의 횡단 방향에서 (등각 게이지는 2개의 자유도를 게이지 고정):

$$
a = \frac{D-2}{2} \sum_{n=1}^\infty n \xrightarrow{\zeta\text{-reg}} \frac{D-2}{2} \cdot \left(-\frac{1}{12}\right) = -\frac{D-2}{24}
$$

$D = 26$에서 $a = -\frac{24}{24} = -1$이므로, 보소닉 끈의 열린 끈에서 $a = 1$. $\square$

### 질량 스펙트럼

| $N$ | $M^2 \alpha'$ | 스핀 | 해석 |
|-----|--------------|------|------|
| 0 | $-1$ | 0 | 타키온 (불안정 진공) |
| 1 | 0 | 1 (열린), 2 (닫힌) | 무질량: 광자, **중력자** |
| 2 | 1 | 0, 1, 2 | 무거운 입자 |
| $N$ | $N-1$ | $\leq 2N$ | 플랑크 질량 근방 탑 |

닫힌 끈의 $N = \tilde{N} = 1$ 준위에 스핀-2 무질량 입자(중력자)가 나타난다. 이것이 다음 문서의 주제다. (→ `04-graviton-necessity.md`)

## 🧪 검증 현실

**중요한 전제:** 끈이론은 직접적인 실험 검증이 없다. 아래의 "검증"은 수학적 구조의 내부 일관성 점검이다.

**✓ 확립: 레게 궤적(Regge trajectory)**

끈의 질량 스펙트럼 $M^2 \propto J$ (스핀 $J$와 질량 제곱의 선형 관계)는 1960년대 강입자 물리 실험 데이터와 정성적으로 일치한다. 이것이 끈 이론의 역사적 기원이다. 그러나 현재는 QCD(색소역학)가 강력의 이론으로 확립되었고, 끈-강입자 대응은 제한적 맥락에서만 유효하다.

**✓ 확립: 영점에너지의 두 방법 일관성**

빛뿔 게이지 양자화와 등각장론 두 방법이 동일한 $a = 1$ (보소닉, 열린 끈)을 준다. 이것은 수학적 일관성의 증거다.

**Python 시연: 끈 진동 스펙트럼**

```python
import numpy as np
import matplotlib.pyplot as plt

# 보소닉 열린 끈의 질량 스펙트럼 시각화
# M^2 * alpha' = N - a, a = 1 (D=26)

alpha_prime = 1.0  # 단위 설정
a = 1.0            # 영점에너지 보정 (보소닉 열린 끈)

# 각 N 준위에서 가능한 스핀과 질량
N_max = 5
levels = []

for N in range(N_max + 1):
    M2 = (N - a) / alpha_prime
    # 최대 스핀: 완전 대칭 텐서 (스핀 = N 또는 N+1에 비례)
    # 열린 끈에서 최대 스핀 = N (N=1: 벡터, N=2: 텐서, ...)
    J_max = N if N > 0 else 0
    label = {0: "타키온", 1: "무질량(광자)", 2: "질량있음"}.get(
        N if N <= 2 else 2, "질량있음")
    levels.append((N, M2, J_max, label))

print("=" * 55)
print(f"{'N':>4} | {'M²α′':>8} | {'최대스핀':>6} | 해석")
print("=" * 55)
for N, M2, J, lbl in levels:
    print(f"{N:>4} | {M2:>8.3f} | {J:>6} | {lbl}")

# 레게 궤적: J vs M^2 (선형 관계 시각화)
# J_max = N = M^2 * alpha' + 1 (보소닉 열린 끈)
M2_range = np.linspace(-1, 4, 200)
J_regge = M2_range + 1  # 레게 궤적

print("\n레게 기울기 α' = 1/T_string")
print(f"선형 관계 J = α' M² + 1 (보소닉 끈)")

# 양자 조화 진동자 사다리 시연
print("\n--- 진동자 사다리 연산자 작용 ---")
# |N> 상태를 생성 연산자로 만들기
# |N> = (α_{-1})^N |0> / sqrt(N!)
def create_state_energy(n_excitations, a=1.0, alpha_prime=1.0):
    """n번 모드를 n_excitations만큼 여기시킨 상태의 M^2"""
    N = n_excitations  # 단일 모드 n=1의 여기수
    return (N - a) / alpha_prime

for n_exc in range(6):
    M2 = create_state_energy(n_exc)
    state_label = f"|N={n_exc}⟩"
    print(f"  {state_label}: M²α' = {M2:+.2f}"
          + (" ← 타키온" if M2 < 0 else " ← 무질량" if M2 == 0 else ""))
```

## 🧬 횡단 원리

**정규 모드와 분리 변수:** 끈의 모드 전개는 편미분 방정식의 변수 분리와 동일한 원리다. 양자 역학에서 무한 사각 우물, 조화 진동자, 음향학에서 기타 줄, 드럼 막의 정규 모드가 모두 같은 수학 구조를 공유한다. (→ L1 oscillations)

**이차 양자화와 입자:** 고전 장을 모드로 분해하고 각 모드를 독립적인 양자 진동자로 취급하는 절차는 QFT의 **이차 양자화(second quantization)**와 동일하다. 끈 이론은 이 절차를 1차원 연장 물체에 적용한 것이다. (→ L3 QFT)

**제타 함수 정규화:** $\sum_{n=1}^\infty n = -1/12$은 리만 제타 함수 $\zeta(-1) = -1/12$의 해석적 연속이다. 이것은 수학에서 잘 정의된 개념이며, 카시미르 효과와 같은 물리적 관측에서도 검증된 기법이다.

## 🪜 창발

각 진동 모드는 서로 다른 스핀, 질량, 게이지 전하를 가진 독립적인 입자처럼 행동한다. 그러나 이 모든 "입자들"은 사실 **하나의 끈**의 서로 다른 상태다. 입자들의 다양성이 끈의 내부 자유도에서 창발한다.

더 심층적으로, D-브레인의 발견으로 열린 끈의 끝점을 고정하는 다차원 물체가 이론에 포함됨이 밝혀졌다. $N$개의 D-브레인을 겹치면 열린 끈 끝점의 내부 지수(Chan-Paton factor)가 $U(N)$ 게이지 대칭을 만들어낸다. 즉, **게이지 대칭 자체가 D-브레인의 기하학으로부터 창발**한다. 이것이 AdS/CFT 대응(→ holographic-principle)의 기반이 되는 통찰이다.

## 📐 예측·반증

**질량 스펙트럼 구조 (이론 내부):**
- $N = 0$: 타키온 (초끈에서 GSO 제거)
- $N = 1$: 무질량 입자 (광자, 중력자)
- $N \geq 2$: $M \sim 1/\sqrt{\alpha'} \sim M_P$ 스케일의 무거운 입자

**검증 불가능한 이유:** 가장 가벼운 무거운 끈 들뜸의 질량이 $\sim M_P \sim 10^{19}\ \text{GeV}$인 반면, 현재 LHC의 최대 에너지는 $\sim 10^4\ \text{GeV}$. 에너지 차이가 $10^{15}$배다.

**반증 가능한 것 (원리적으로):**
- 만약 $\alpha'$가 매우 크다면(끈이 매우 부드럽다면) $M \sim 1/\sqrt{\alpha'}$가 낮아져 LHC 범위에 들어올 수 있다. 일부 대형 여분 차원 모형은 이런 가능성을 제안했으나 LHC 데이터로 제한된다.

🔬 **Boundary** — ✓ 확립: 모드 전개·양자화·질량 공식 $M^2=(N-a)/\alpha'$ $\square$, 닫힌 끈 스핀-2 무질량 입자. ✗ 미배달: $\alpha'$ 값 제1원리 예측, 표준모형 스펙트럼 재현, 플랑크 스케일 아래 실험 검증.

🪜 **Emergence** — 하나의 끈의 내부 진동으로부터 다양한 입자들이 창발하고, D-브레인 기하학에서 $U(N)$ 게이지 대칭이 창발하며, 이것이 AdS/CFT에서 경계 게이지 이론이 되는 구조로 이어진다.
