# 🧵 String Theory Distilled

### *"끈은 진동한다"* 고 **외우는 것** 과,

$$\text{닫힌 끈의 무질량 스핀-2 모드} \;\;\Longrightarrow\;\; g_{\mu\nu}\;(\textbf{중력자})$$

### 이 진동 모드가 **왜 필연적으로 중력자** 이고 — 끈이론이 중력을 *선택* 이 아니라 *요구* 하는지,

### 쌍대성 $R \leftrightarrow \alpha'/R$ 이 **왜 "근본은 관점 의존"** 임을 드러내는지를 아는 것은 **다르다.**

<br/>

> *끈이론을 **"만물의 이론"** 으로 **떠받드는 것** 과,*
>
> $$S_{\text{BH}} = \frac{A}{4\ell_P^2} \;\;\xleftarrow{\;\text{미시상태 계수}\;}\;\; \text{D-브레인 BPS 상태 수 (스트로민저–바파)}$$
>
> *끈이론이 블랙홀 엔트로피를 **실제로 센** 가장 단단한 성취와, **그것이 특수한 BPS 경우** 라는 정직한 한계를 아는 것은 다르다.*
>
> *AdS/CFT 를 **신기해하는 것** 과,*
>
> $$Z_{\text{gravity}}[\text{AdS}_{d+1}] \;=\; Z_{\text{CFT}}[\partial(\text{AdS}_d)], \qquad S_A = \frac{\text{Area}(\gamma_A)}{4G_N}\ (\text{류–타카야나기})$$
>
> *차원 하나가 **창발** 하는 홀로그래피의 구체적 실현과, **그러나 우리 우주는 AdS 가 아니다** 라는 미해결을 아는 것은 다르다.*
>
> *끈이론을 **"검증 불가한 수학 유희"** 로 **비웃는 것** 과,*
>
> $$N_{\text{진공}} \sim 10^{500} \;\;\Longrightarrow\;\; \text{예측력의 위기 — "끈이론은 과학인가?"}$$
>
> *$10^{500}$ 진공의 풍경이 **왜** 반증 가능성의 위기인지, 그럼에도 무엇이 **부정할 수 없는 성취** 인지를 가르는 것은 다르다.*

<br/>

**끈이론은 가장 발전했으나 아직 미배달인 이론이다** — 떠받들지도 비웃지도 않고, **무엇이 성취이고 무엇이 미배달인지** 를 과대포장 없이 정직하게 가르는 것이 곧 이해다.

<br/>

**다루는 개념 (점에서 끈으로의 계보순)**

남부–고토/폴랴코프 작용 · 세계면(worldsheet) · 진동 모드 = 입자 스펙트럼 · 무질량 스핀-2 = 중력자 · 임계 차원(10D) · 이상 상쇄 · 칼라비–야우 콤팩트화 · 초대칭(SUSY) · 다섯 끈이론 · T-쌍대성 · S-쌍대성 · M이론 · D-브레인 · 게이지/중력 이중성 · 블랙홀 미시상태 · 스트로민저–바파 · 퍼지볼 · 정보 역설 · 말다세나 추측(AdS/CFT) · 홀로그래피 사전 · 류–타카야나기 · 얽힘 = 기하 · 끈 풍경 · 인류원리 · 반증 가능성

<br/>

**핵심 질문**

> **무엇이 성취이고, 무엇이 아직 미배달인가 — 그리고 끈이론은 과학인가?** 모든 주장에는 *확립(중력자·블랙홀 엔트로피·AdS/CFT)*, *유망(게이지/중력 이중성의 응용)*, *미배달(표준모형 유도·풍경의 예측력·실험 검증)* 의 경계가 있다. 이 레포는 **"만물의 이론"** 도 **"헛소리"** 도 금지하고, 닫힌 끈의 스핀-2 모드가 *왜 중력자인지* 부터 $10^{500}$ 풍경의 *예측력 위기* 까지 — **끈의 작용 → 진동 모드 → 쌍대성 → 홀로그래피 → 정직한 경계** 를 한 줄씩 유도합니다. 그리고 성취가 멈추는 바로 그곳에 *다음 질문* 이 있음을 위 레이어로 화살표 합니다.

## 🎯 이 레포에 대하여

끈이론을 다루는 자료는 대부분 양극단에 빠집니다 — **"모든 것을 통일하는 만물의 이론"** 이라는 과대포장, 또는 **"검증 불가한 수학 유희"** 라는 냉소. 하지만 닫힌 끈의 무질량 스핀-2 모드가 *왜* 필연적으로 중력자인지(→ 중력이 *요구* 됨), T·S·AdS/CFT 쌍대성이 *왜* "근본은 관점 의존"임을 드러내는지(→ 시공간 창발의 암시), 스트로민저–바파가 블랙홀 엔트로피를 *어떻게* 미시상태로 세는지(→ 가장 단단한 성취), 그리고 $10^{500}$ 풍경이 *왜* 예측력의 위기인지 — 이런 "왜"는 어느 쪽도 정직하게 설명하지 않습니다.

이 레포는 끈이론을 **성취와 미배달의 통제 가능한 분류** 로 환원합니다. 모든 주장에 **무엇이 확립 · 무엇이 유망 · 무엇이 미배달 · 무엇이 반증 가능** 한지를 명시하고, sympy 기호 전개와 numpy/scipy 수치 시연으로 *수학적 구조와 내부 일관성* 을 눈으로 검증합니다.

> ⚠️ **정직성 선언**: 끈이론은 *직접적 실험 검증이 없다.* 이 레포의 검증은 입자 가속기 데이터가 아니라 모드 스펙트럼·쌍대성 망·홀로그래피 공식의 *수학적 일관성* 점검입니다. 우리는 이 사실을 숨기지 않으며, 매 문서에서 성취(✓)와 미배달(✗)을 명확히 가릅니다.

| 일반 자료 | 이 레포 |
|----------|---------|
| "끈은 진동하고 그 모드가 입자다" | **모드 = 스펙트럼의 메커니즘** — 세계면 위 진동 정규모드 $M^2 = (N-a)/\alpha'$. $n=1$ 무질량 준위에 게이지장과 **닫힌 끈의 스핀-2 모드** 출현. 사다리형 질량 스펙트럼을 numpy 로 시연(→ L1 oscillations) $\square$ |
| "끈이론은 중력을 포함한다" | **중력자의 필연** — 닫힌 끈의 무질량 스핀-2 대칭 텐서가 일관성(게이지 불변)을 위해 *반드시* 아인슈타인 방정식을 따라야 함. 중력은 *선택이 아니라 요구* (→ L4 GR) $\square$ |
| "여분 차원이 필요하다" | **임계 차원 10D** — 바일 이상(anomaly) 상쇄가 시공간 차원을 *고정*. $D=26$(보존끈)·$D=10$(초끈)이 *유도되는* 결과임을 중심전하 $c$ 계산으로(→ L0 symmetry, L3) $\square$ |
| "쌍대성은 신기하다" | **근본은 관점 의존** — T-쌍대성 $R\leftrightarrow\alpha'/R$(최소 길이 $\sqrt{\alpha'}$), S-쌍대성 $g\leftrightarrow1/g$(강↔약), M이론이 다섯 끈이론을 통일. "무엇이 근본인가"가 기술에 의존(→ emergent-spacetime) $\square$ |
| "블랙홀 엔트로피를 설명한다" | **미시상태를 *센다*** — 스트로민저–바파: 특정 BPS 블랙홀의 D-브레인 구성 상태 수 $\to S=A/4\ell_P^2$ *정확히 재현*. **정직**: 일반(비-BPS) 블랙홀로의 확장은 미완(→ L4 black-holes, L2) $\square$ |
| "AdS/CFT 는 홀로그래피다" | **차원 하나가 창발** — $(d{+}1)$차원 AdS 중력 ≡ $d$차원 경계 CFT. 강결합 CFT를 약중력으로 계산. 류–타카야나기 $S_A=\text{Area}/4G_N$. **정직**: 우리 우주는 AdS가 아니다(de Sitter)(→ holographic-principle) $\square$ |
| "풍경은 다 설명한다" | **예측력의 위기** — 콤팩트화 선택 $\sim10^{500}$ 진공. 무엇이든 "설명" 가능 → *아무것도 예측 못 함*. 인류원리는 설명인가 포기인가의 논쟁(→ L4 cosmology, L0 measurement) |
| "끈이론은 반증 불가다" | **포퍼를 정면으로** — 직접 실험 검증의 부재를 숨기지 않고, *무엇이 끈이론을 반증할 수 있는가* 를 정직하게 분류. 과학철학의 경계 사례로(→ L0 measurement) |
| 개념 나열 | numpy/scipy 로 **끈 진동 스펙트럼 사다리** · **T-쌍대성 스펙트럼 불변** · **류–타카야나기 얽힘=면적** · **AdS/CFT 사전 $\Delta(\Delta{-}d)=m^2$** · **성취/미배달 분류표** 를 직접 구현해 수학적 주장을 눈으로 확인 |

# 환경: Python 3 + numpy + scipy + matplotlib
import numpy as np

# ① 끈 진동 스펙트럼 (Ch1-03, Ch7-01)
def string_spectrum(n_max, alpha=1.0, a=1.0):
    # 질량²  M² = (N − a)/α'  (N: 진동 준위, a: 영점에너지)
    N = np.arange(0, n_max)
    return (N - a) / alpha           # 보존끈 a=1; 타키온·무질량·대량 모드
# → n=1 에 무질량 모드(중력자·게이지장 포함), 사다리형 질량 (→ L1 oscillations)

# ② T-쌍대성 장난감 (Ch3-02, Ch7-02)
def t_dual_spectrum(R, n, w, alpha=1.0):
    # 운동량 모드 n/R 와 감김 모드 wR/α'
    return (n / R)**2 + (w * R / alpha)**2
# → R ↔ α'/R 교환 시 (n↔w) 스펙트럼 불변 확인, 최소 길이 √α' (→ quantum-gravity)

# ③ AdS/CFT 사전 맛보기 (Ch5-02, Ch7-05)
def conformal_dimension(m2, d):
    # 경계 연산자 차원 Δ ↔ 벌크 장 질량 m: Δ(Δ−d) = m²
    return (d + np.sqrt(d**2 + 4 * m2)) / 2
# → 벌크-경계 대응의 가장 단순한 사전 (→ holographic-principle)

# ④ 류–타카야나기 (Ch5-04, Ch7-03)
def rt_entropy(ell, eps, c):
    # AdS₃/CFT₂: 경계 구간의 얽힘 엔트로피 ∝ 측지선 길이
    return (c / 3) * np.log(ell / eps)     # S_A = (c/3) ln(ℓ/ε) 재현 (→ L3 quantum-information)

# ⑤ 콤팩트화 차원 세기 (Ch7-04)
#    10 = 4(관측) + 6(콤팩트) 의 모듈라이 수 맛보기
#    → 풍경의 기원: 콤팩트화 선택의 방대함 (~10⁵⁰⁰) 시연

# ⑥ 성취/미배달 분류표 (Ch6-05, Ch7-06)
